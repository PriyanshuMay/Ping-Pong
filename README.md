## Setup

```bash

git clone https://github.com/ahampriyanshu/gonewz.git && cd gonewz
echo NEWS_API_KEY="xxxxxxxxxxxx" >> .env
go build && ./gonewz

```


## Dependencies

| Dependency | Version |
| --- | --- | 
| Go | 1.15 | 
| Tailwind | 2.0 |
